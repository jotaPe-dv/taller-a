# Taller A

Repositorio con los ejercicios y evidencias de los subtalleres del taller de almacenamiento y procesamiento de grandes datos.

## Subtalleres

- **Subtaller 1:** configuración de la cuenta de AWS y creación de llaves de acceso en IAM.
- **Subtaller 2:** repaso de SQL con Python usando `sqlite3`, con notebook y PDF de impresión.
- **Subtaller 3:** ejercicio con metodología Medallion en Apache Spark, con notebook, PDF de impresión y capturas de la migración a AWS en un documento DOCX.

## Estructura

- [subtaller1](subtaller1): material relacionado con la configuración de AWS.
- [subtaller2](subtaller2): notebook de SQL con Python y su PDF.
- [subtaller3](subtaller3): notebook de Spark Medallion, PDF y documento de capturas.
